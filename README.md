# 该仓库用于记录c++学习过程中的重点
##day01
    **命名空间**
    最好是不要一次性全部引入，用什么引入什么。

    **跨模块调用**
    1.用于代码结构简单的。
    2.静态变量只在本模块起作用

    **const常量和宏定义常量的区别**
    1.时机不同，宏定义时在预处理时进行字符串替换,const是在编译时。
    2.宏定义没有类型。

    **const与指针**
    1.const int *p(常量指针)  const在*号前：不能改指针所指向的值,但是可以更改指向
    2.int * const p(指针常量) const在*号后：不能改指向，但是可以改指向的值
